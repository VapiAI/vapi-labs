## Browsing / running a project

Every project folder is self-contained and includes:

- `README.md` — what it does, how to run it, what it needs
- `.env.example` — required environment variables (never real keys)
- Optionally `AGENTS.md` and/or `SKILLS.md`

Clone the repo, `cd` into a project folder, and follow its README.
